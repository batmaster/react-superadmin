# Button Component

The Button component provides various button styles and sizes for your admin interface with built-in loading states and accessibility features.

## Basic Usage

```tsx live
import React from 'react';
import { Button } from '@react-superadmin/web';

function ButtonExample() {
  return (
    <div className='space-y-4'>
      <Button variant='primary'>Primary Button</Button>
      <Button variant='secondary'>Secondary Button</Button>
      <Button variant='outline'>Outline Button</Button>
      <Button variant='ghost'>Ghost Button</Button>
      <Button variant='danger'>Danger Button</Button>
    </div>
  );
}
```

## Loading State

```tsx live
import React, { useState } from 'react';
import { Button } from '@react-superadmin/web';

function LoadingButtons() {
  const [loading, setLoading] = useState(false);

  const handleClick = () => {
    setLoading(true);
    setTimeout(() => setLoading(false), 2000);
  };

  return (
    <div className='space-x-2'>
      <Button onClick={handleClick} loading={loading}>
        {loading ? 'Loading...' : 'Click Me'}
      </Button>
    </div>
  );
}
```

## API Reference

| Prop        | Type                                                           | Default     | Description                               |
| ----------- | -------------------------------------------------------------- | ----------- | ----------------------------------------- |
| `variant`   | `'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'danger'` | `'primary'` | Button style variant                      |
| `size`      | `'sm' \| 'md' \| 'lg'`                                         | `'md'`      | Button size                               |
| `loading`   | `boolean`                                                      | `false`     | Shows loading spinner and disables button |
| `disabled`  | `boolean`                                                      | `false`     | Whether button is disabled                |
| `onClick`   | `function`                                                     | -           | Click handler                             |
| `className` | `string`                                                       | -           | Additional CSS classes                    |
| `children`  | `ReactNode`                                                    | -           | Button content                            |

## CSS Classes

The Button component uses Tailwind CSS classes for styling:

- **Primary**: `bg-primary-600 text-white hover:bg-primary-700 focus:ring-primary-500`
- **Secondary**: `bg-gray-600 text-white hover:bg-gray-700 focus:ring-gray-500`
- **Outline**: `border border-gray-300 bg-white text-gray-700 hover:bg-gray-50 focus:ring-primary-500`
- **Ghost**: `text-gray-700 hover:bg-gray-100 focus:ring-gray-500`
- **Danger**: `bg-red-600 text-white hover:bg-red-700 focus:ring-red-500`

## Accessibility

Buttons include proper accessibility features:

- Semantic HTML (`<button>` element)
- Proper ARIA labels when needed
- Keyboard navigation support
- Focus indicators with `focus:ring-2 focus:ring-offset-2`
- Screen reader support
- Loading state management
- Disabled state handling

## Usage Examples

### Form Submission

```tsx
import { Button } from '@react-superadmin/web';

function SubmitForm() {
  const [isSubmitting, setIsSubmitting] = useState(false);

  const handleSubmit = async () => {
    setIsSubmitting(true);
    try {
      await submitData();
    } finally {
      setIsSubmitting(false);
    }
  };

  return (
    <Button type='submit' loading={isSubmitting} disabled={isSubmitting}>
      {isSubmitting ? 'Saving...' : 'Save Changes'}
    </Button>
  );
}
```

### Action Buttons

```tsx
import { Button } from '@react-superadmin/web';

function ActionButtons() {
  return (
    <div className='space-x-2'>
      <Button variant='outline' onClick={handleCancel}>
        Cancel
      </Button>
      <Button variant='danger' onClick={handleDelete}>
        Delete
      </Button>
      <Button variant='primary' onClick={handleSave}>
        Save
      </Button>
    </div>
  );
}
```
